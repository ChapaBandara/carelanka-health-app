# CareLanka - Digital Health & Medication Compliance App

A Flutter/Firebase personal healthcare app designed to help users manage medications, track health records, and stay on top of appointments. Currently in active development.

## Features
- Medication tracking with AI-driven smart reminders
- Health record management & document library
- Appointment scheduling & tracking
- Family profile management with QR linking
- Allergy & illness tracking
- Drug conflict detection
- Daily, weekly, monthly & yearly health reports
- Cross-device authentication & real-time Firebase sync
- Notification preferences & privacy settings

## Tech Stack
- **Framework:** Flutter / Dart
- **Backend:** Firebase (Auth, Firestore, Cloud Messaging)
- **Design:** Figma
- **Architecture:** Provider pattern
- **AI:** Rule-based recommendation engine

## Project Status
🚧 **In Progress** — actively being developed as part of academic research

```bash
   git clone https://github.com/ChapaBandara/carelanka-health-app.git
```

3. Run the app
